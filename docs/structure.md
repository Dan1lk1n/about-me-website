## Компоненты (reusable)

- `Component/Section Header` — заголовок секции (eyebrow + title + subtitle)
- `Component/Project Card` — карточка проекта (иконка, название, описание, роль)
- `Component/Tool Card` — карточка инструмента (иконка, название, описание)
- `Component/Contact Link` — ссылка контакта (иконка + текст)
- `Component/Stack Category` — категория стека (label + значения)
